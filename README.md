unity-yaml2lua
==============

> Parse unity yaml file to lua table.

```
Usage: unity-yaml2lua [-o <output>] [-h] <input>

Parse unity yaml file to lua table.

Arguments:
   input                 Input yaml file.

Options:
   -o <output>, --output <output>
                         Output lua file.
   -h, --help            Show this help message and exit.
```
